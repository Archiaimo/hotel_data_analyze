总结
*开房人数中，男女比列大概2：1
*主要开房人群集中在我国华东区域，这与华东区域人口密集，经济发达有关。
*经济强省江苏，浙江，山东开房人数较多
*按照年龄划分，80后，70后占主导。可以推断，90后在近几年将会紧随其后。80后马上四十不惑，开房次数仍然占据主导，个中原因，值得深思。
*星座人数最多的是天秤，天蝎，处女
*生肖最多的是兔虎龙，最少的是羊
*手机运营商里移动占据绝对的市场份额
